## Features (v1.6.0)

- Interactive directory navigation (go, godrive, back, gos, gosc)
- Directory tree display with configurable depth
- **NEW!** JSON, Markdown, and HTML report generation
- **NEW!** Clipboard copy for reports (`--cp`)
- **NEW!** Full Linux/WSL support (.deb package available)
- File content viewing with optional line numbers
- Ignore/care system for directories and extensions
- Persistent configuration
- Teleport savepoints (bookmarks)
- Run aliases for custom commands

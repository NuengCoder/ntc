# ntc - Navigate, Tree, Cat > README.md

A combined directory tree viewer and file concatenator.

## Features

- Interactive directory navigation (go, godrive, back)
- Directory tree display with configurable depth  
- TXT and HTML report generation >> README.md
- File content viewing with optional line numbers 
- Ignore/care system for directories and extensions  
- Persistent configuration

## Quick Start 

ntc --version 
ntc --help 
ntc -i src                    # Generate TXT report 
ntc -i src -o report.html     # Generate HTML report 
ntc                           # Launch interactive mode
